Huawei Message Receipt
Huawei has a message delivery report feature called
Message Receipt
After a notification sends to device, Huawei will report the message status to Netmera. By this way, Netmera can track uninstalled devices. To be able to use this feature, you must integrate Message Receipt in Huawei App Gallery.
Open
App Gallery Connect
(first you need to login to your
HMS
account)
Go to
section
Click
Settings
tab
Click
Enable
button for
App Receipt
Click
New Button
Enter configurations.
Callback address
and
HTTPS certificate
must be as below. After you check the details, click
Submit
button.
Callback Address
https://report.netmera.com/reporter/hms/message-receipt
