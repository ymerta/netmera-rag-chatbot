Delete User
The
deleteUsersPermanently
endpoint is used to
permanently delete all user data
from the Netmera system. This includes all associated events, attributes, and records for the specified user(s). Once deleted, the data cannot be recovered.
This endpoint is only available to accounts with
data deletion access
, which must be requested through the Netmera panel.
Endpoint
POST /rest/3.0/deleteUsersPermanently
Request Headers
X-netmera-api-key
: Your REST API key
Content-Type
application/json
Example Request
curl -X POST https:
//restapi.netmera.com/rest/3.0/deleteUsersPermanently \
"X-netmera-api-key: your_rest_api_key"
"Content-Type: application/json"

"extIds"
